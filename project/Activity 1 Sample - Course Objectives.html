Program Leading with Clarity: A Manager Essentials Series
Audience First-year people managers, individual contributor promotions
Format · Duration 4-module blended program · 8 hours total · 90-day arc
Terminal Objective — The whole program
By the end of this program, participants will be able to manage the day-to-day performance and development of a direct report team using structured feedback, coaching conversations, and clear expectations — demonstrating measurable improvement in team engagement scores and direct report performance within 90 days.

Module 1 — Setting Clear Expectations

Duration: 90 minutes · Format: Instructor-led workshop · Assessment: Role-play with observer feedback

Enabling Objectives

1.1

Given a new or shifting project, participants will be able to write a clear expectations brief that specifies deliverables, standards, timeline, and decision authority — in 30 minutes or fewer.

Bloom's: Apply Method: Job aid + practice Assessment: Brief review + peer critique
1.2

Participants will be able to distinguish between an ambiguous expectation and a clear one, identifying at least 3 gaps in a provided example brief before revision.

Bloom's: Analyze Method: Case study Assessment: Gap-identification exercise
1.3

Participants will be able to deliver an expectations conversation using a structured format (context → deliverable → standard → check for understanding) in a role-play scenario rated as "clear" by the observer.

Bloom's: Apply Method: Observed role-play Assessment: Observer checklist

Module 2 — Delivering Effective Feedback

Duration: 2 hours · Format: Blended (30-min async pre-work + 90-min live session) · Assessment: Observed feedback conversation

Enabling Objectives

2.1

Participants will be able to construct a feedback statement using the SBI model (Situation–Behavior–Impact) for both positive and developmental feedback scenarios, with specific behavioral evidence.

Bloom's: Apply Method: Model practice + cases Assessment: Written feedback drafts
2.2

Participants will be able to identify and correct 4 common feedback errors (vague language, judgment instead of behavior, sandwich framing, no follow-up) when reviewing sample feedback transcripts.

Bloom's: Evaluate Method: Transcript critique Assessment: Error identification exercise
2.3

In a live role-play, participants will be able to deliver a developmental feedback conversation that includes an SBI statement, an invitation for the direct report to respond, and a mutually agreed next step.

Bloom's: Apply Method: Live role-play Assessment: Observer rubric + self-assessment
2.4

Participants will be able to explain why delivering positive feedback with specificity improves retention and performance, citing at least one mechanism from the session content.

Bloom's: Explain Method: Discussion Assessment: Post-discussion reflection prompt

Module 3 — Coaching Through Performance Gaps

Duration: 2.5 hours · Format: Instructor-led + simulation · Assessment: Coaching conversation simulation

Enabling Objectives

3.1

Participants will be able to distinguish between a performance issue that warrants coaching and one that warrants a direct instruction or corrective action, given 5 scenario descriptions.

Bloom's: Analyze Method: Scenario sort Assessment: Decision accuracy (4 of 5 correct)
3.2

Participants will be able to conduct a 10-minute coaching conversation using a structured framework (clarify the gap → explore root cause → identify options → commit to action), without redirecting to "here's what I would do."

Bloom's: Apply Method: Coaching simulation Assessment: Observer checklist + self-rating
3.3

Participants will be able to prepare for a "below expectations" performance conversation, including documentation of the pattern, a draft of the conversation opening, and a proposed follow-up plan.

Bloom's: Apply Method: Prep exercise + peer review Assessment: Preparation checklist review

Module 4 — Building the Habit: 30-Day Practice Plan

Duration: 2 hours · Format: Action planning workshop + 30-day check-in structure · Assessment: 30-day behavior self-report

Enabling Objectives

4.1

Participants will be able to build a personal 30-day practice plan that identifies 3 specific manager behaviors to build, the situations they'll practice them in, and a tracking method that takes fewer than 5 minutes per week.

Bloom's: Create Method: Action planning workshop Assessment: Plan review by facilitator
4.2

Participants will be able to identify 2 accountability structures (a peer manager, a mentor, a check-in cadence) that will support their practice over the 30-day period.

Bloom's: Apply Method: Partner matching activity Assessment: Accountability commitments sheet

Assessment Summary

Module Assessment Method Passing Standard Evaluator
1 — Setting Expectations Expectations brief + observed role-play Observer rates conversation as "clear" on 3 of 4 criteria Peer observer + facilitator
2 — Delivering Feedback Written feedback drafts + live role-play SBI model correctly applied; observer rubric score ≥ 3/4 Facilitator + peer
3 — Coaching Scenario sort + coaching simulation Scenario sort: 4 of 5 correct; simulation: no redirecting to advice Facilitator
4 — Practice Plan 30-day plan + accountability commitments Plan includes 3 behaviors, 3 situations, and a tracking method Facilitator review
